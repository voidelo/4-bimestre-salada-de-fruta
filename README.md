# 4-bimestre-salada-de-fruta

RELATORIO DE DW

Meu projeto finalizou com cinco CRUDs, sendo eles: cargo, pessoa, produto, pedido e endereço. Esses CRUDs permitiram compreender melhor a estrutura de um sistema web e a integração com banco de dados. 

Com o projeto, aprendi melhor a organizar arquivos no Visual Studio Code, entender a divisão entre frontend e backend e como funciona a conexão com o banco de dados. Apesar disso, tive muita dificuldade com lógica de programação e ainda não me sinto segura para desenvolver muitas partes do sistema sozinha.

A maior dificuldade foi no backend. Mesmo com ajuda da IA, muitas vezes o código não funcionava como esperado. Para superar isso, analisei projetos de colegas, testei várias soluções e fui ajustando até conseguir um resultado funcional.

Também tive dificuldade em usar a Inteligência Artificial, pois nem sempre consegui criar prompts claros. Aprendi que a IA funciona melhor quando explicamos o problema passo a passo e em partes menores.

Eu esperava conseguir entregar o projeto completo, porém não consegui finalizar tudo como planejei. Mesmo o resultado não sendo como o planejado, creio que compreendi coisas essenciais com a disciplina ao longo do ano.


A IA foi muito útil, mas também causou frustração quando não entregava exatamente o que eu esperava. Com a experiência, aprendi a utilizá-la melhor.

Tenho mais preferência pelo frontend, apesar de também ser desafiador. O backend foi a parte mais difícil, mas onde mais aprendi.


O projeto de DW1 foi importante para meu aprendizado, mesmo com dificuldades. Ele contribuiu para entender melhor o funcionamento de sistemas web e mostrou pontos que ainda preciso melhorar nas próximas disciplinas.
